# Contributing to ConfigShape

Thanks for your interest in contributing!

## Development setup

```bash
pnpm install
pnpm dev          # web playground at http://localhost:5173
pnpm test         # core unit tests
pnpm build        # build all packages
```

## Pull requests

1. Fork the repo and create a feature branch from `main`.
2. Write tests for core logic changes.
3. Ensure `pnpm test` and `pnpm build` pass.
4. Use conventional commit messages (`feat:`, `fix:`, `test:`, etc.).
